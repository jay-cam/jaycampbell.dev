# jaycampbell.dev

This repository contains coursework and supporting files for CSE 135 (Web Servers and Web-Based Systems).

The project is deployed to a personal server and evolves across multiple homework assignments focused on understanding web servers, HTTP, CGI, analytics, and server-side behavior.

## Homework 1
- Server setup and deployment on a VPS
- Apache configuration and HTTPS
- Custom error handling
- Server identity and security hardening
- Log inspection and analysis

## Homework 2
- CGI programming in multiple languages (Perl, Python, Go, Node.js)
- Handling different HTTP methods and encodings
- Server-side state management
- Integration and comparison of analytics tools

**Some assignment documentation and operational details are intentionally excluded from this repository to avoid exposing deployment or credential information.**
